(A) A adalah himpunan bagian dari A itu sendiri (yaitu, A ? A )
(B) Himpunan kosong merupakan himpunan bagian dari A ( ? ? A)

Pembuktian dengan menggunakan diagram ven
Buatlah diagram venn untuk bagian ruas kiri keramaan dan diagram venn untuk ruas kanan kesamaan. Jika diagram venn keduanya sama, berarti kesamaan tersebut benar.

Contoh:
Misalkan A, B, dan C adalah himpunan. Buktikan A ? ( B ? C) dengan diagram venn.
Penyelesaian:
Diangram venn untuk ruas kiri dan ruas kanan masing-masing ditunjukan pada gambar di bawah , dari diagram venn untuk masing-masing ruas di atas, keduanya memberikan area arsiaran yang sama jadi, terbukti A ? (B ? C) = (A ? B)?( A ? C )


Dengan diagram venn, pembuktian dapat dilakukan dengan cepat. Ini adalah kelebihan metode ini. Namun, kekurangannya, diagram venn hanya dapat digunakan jika himpunan yang digambarkan tidak banyak jumlahnya. Selain itu, metode ini mengilustrasikan ketimbang membuktikan fakta.
Banyak matematikanwan tidak menganggapnya sebagai metode yang valid untuk pembuktian secara formal. Oleh karena itu, pembuktian dangan diagram venn kurang dapat diterima

2. P ? Q adalah suatu himpunan ganda yang multiplisitas elemennya sama dengan multiplisitas minimum elemen tersebut pada himpunan P dan Q.
Contoh: P = { a,a,a,c,d,d } dan Q = { a,a,b,c,c } maka P ? Q = { a,a,c }
3. P – Q adalah suatu himpunan ganda yang multiplisitas elemennya sama dengan:
- Multiplitas elemen tersebut pada P dikurangi multiplisitasnya pada Q, jika selisihnya positifnya
- 0, jika selisihnya nol atau negatif.
Contohnya: jika P = { a,a,a,b,b,c,d,d,e } dan Q = { a,a,b,b,b,c,c,d,d,f } maka P – Q = { a,e }
