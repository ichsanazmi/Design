    Soal Induksi Matematika

    1. Buktikan melalui induksi matematik bahwa:

    a. 1(2) + 2(3) + ... + n(n + 1) = n(n+1)(n+2)/3 untuk semua n = 1

    b. 1/(1(2)) + 1/(2(3)) + 1/(3(4)) + ... + 1/(n(n+1)) = n/(n +1) untuk semua n = 1

    c. 12 + 32 + 52+ ... + (2n 1)2 = (n(2n-1)(2n+1))/3 untuk semua n = 1

    d. 1 + a + a2 + ... + an = (1-a^(n+1))/(1- a) untuk semua n = 0 dan a ? 1

    e. 3 + 3.5 + 3.52 + ... + 3.5n = (3(5^(n+1)-1))/4 untuk semua n = 0

    2. Buktikan melalui induksi matematik bahwa n4 - 4n2 habis dibagi 3 untuk semua bilangan bulat n = 2.

    3. Buktikan dengan induksi matematik bahwa sejumlah pangkat tiga dari tiga buah bilangan bulat positif berurutan selalu habis dibagi 9.

    4. Buktikan bahwa surat pos yang menggunakan perangko 24 sen atau lebih dapat hanya menggunakan 5 sen atau 7 sen.

    5. Untuk biaya pos berapa saja dapat menggunakan perangko 5 sen dan perangko 6 sen? Buktikan jawaban anda dengan menggunakan induksi matematik!

    6. Sebuah kios penukaran uang hanya mempunyai pecahan uang senilai senilai Rp 2.000 dan Rp 5.000. Untuk uang senilai berapa saja yang dapat ditukar dengan kedua pecahan tersebut? Buktikan jawaban anda dengan induksi matematik.
